## Setup Instructions

### 1. Replace Placeholder Content
- Update phone numbers, email, and location in all HTML files
- Replace `yourname@vestige.com` with your email
- Update `+91 98765 43210` with your phone number
- Add your city name in location fields
- Replace placeholder images in `assets/images/` folder

### 2. Add Your Images
Place your images in these folders:
- `assets/images/products/` - Product photos
- `assets/images/banners/` - Banner images
- `assets/images/profile/` - Your photos
- `assets/images/icons/` - Logo and icons

### 3. Add PDF Files
Place your PDF files in:
- `assets/pdf/price-list.pdf` - Product price list
- `assets/pdf/business-plan.pdf` - Business plan document
- `assets/pdf/product-catalog.pdf` - Product catalog

### 4. Deployment

#### Option A: Deploy to Vercel (Recommended)
1. Push code to GitHub repository
2. Go to [vercel.com](https://vercel.com)
3. Import your GitHub repository
4. Vercel will automatically deploy your site

#### Option B: Deploy to GitHub Pages
1. Rename repository to `yourusername.github.io`
2. Enable GitHub Pages in repository settings
3. Set source to `main` branch
4. Your site will be at `https://yourusername.github.io`

#### Option C: Deploy to Netlify
1. Push code to GitHub repository
2. Go to [netlify.com](https://netlify.com)
3. Drag and drop your folder or connect GitHub
4. Netlify will deploy your site
